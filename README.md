# MORELSTACKHOUSE.COM

Gatsby repo for morelstackhouse.com

## Develop

```shell
gatsby develop
```

## Build

```shell
gatsby build
```

## Features and Dependencies

- Update menu items by editing the menu array in `./gatsby-config.js`
- Update Bootstrap variables in `./src/components/theme/_variables.scss`
